Introduction¶

The RANK API Server provides Bloomberg users with the ability to extract RANK data from Bloomberg. The RANK API Server for extracing RANK data requires separate authorization by Bloomberg.

Note

RANK API users will need the following steps completed before using the RANK API service.

  1. Signed RANK API Agreement.
  2. Install serverapi.exe and register with Bloomberg for server access to the RANK API.
  3. Download Bloomberg Desktop API v3 SDK from WAPI<GO> in Bloomberg terminal.

To get access to RANK API in UAT and production, please reach out to your Bloomberg EMSX / RANK representitive.

Support¶
